# 착한 남자 착각 시스템 진단 퍼널

## 📋 프로젝트 개요

**『프레임 전쟁』** 전자책 마케팅을 위한 연애 진단 퍼널 시스템입니다.  
사용자의 연애 패턴을 5가지 질문으로 진단하고, 3가지 레벨(LEVEL 1, 2, 3)로 분류하여 맞춤형 솔루션과 전자책 챕터를 추천합니다.

---

## 🎯 주요 기능

### 1. 5단계 진단 시스템
- **5가지 핵심 질문**으로 사용자의 연애 패턴 분석
- 점수 기반 자동 레벨 분류 (0~25점)
- 실시간 프로그레스 바 및 질문 번호 표시

### 2. 정교한 레벨 판정 시스템

#### **점수 기준**
- **LEVEL 1 (0~10점)**: 고착화된 프렌드존 패턴
- **LEVEL 2 (11~19점)**: 착한 남자 프레임 적신호
- **LEVEL 3 (20~25점)**: 프레임 설계자 가능성

#### **특별 규칙: LEVEL 3 차단**
**최악 답변(0점 또는 1점)이 하나라도 있으면:**
- 아무리 총점이 높아도 LEVEL 3 불가
- 자동으로 LEVEL 2로 강등
- **목적**: 진짜 우수한 사람만 LEVEL 3 부여

### 3. 레벨별 맞춤형 답변 인용

#### **LEVEL 1 & 2: 최악 답변 인용**
```javascript
getWorstAnswers() 사용
→ 0~2점 답변 중 최대 2개
→ 점수 낮은 순 정렬
```

**예시:**
> 당신은 "새벽에 연락하면?" 질문에  
> **"아무리 멀어도 바로 달려간다"**(0점)라고 답했고,
>
> "2시간 지각" 질문에  
> **"별 말 없이 넘어간다"**(0점)라고 답했습니다.

#### **LEVEL 3: 최상위 답변 인용**
```javascript
getBestAnswers() 사용 (신규)
→ 4~5점 답변 중 최대 2개
→ 점수 높은 순 정렬
```

**예시:**
> 당신은 "새벽에 연락하면?" 질문에  
> **"힘들다며 다른 날을 제안한다"**(4점)라고 답했고,
>
> "답장 늦음" 질문에  
> **"다른 일을 하며 답이 오면 천천히 확인한다"**(5점)라고 답했습니다.

### 4. 심리적 흐름 최적화

각 레벨별로 차별화된 심리 흐름:

```
1️⃣ 당신의 현재 상태 (진단 결과 + 답변 인용)
   ↓
💙 공감 및 어루만지기 (자존감 보호)
   ↓
🤔 의문 해소 및 반론 차단 (통합 스크립트)
   ↓
2️⃣ ⚠️ 이대로 두면 어떻게 될까요?
   ↓
✨ 희망 메시지
   ↓
3️⃣ 💡 맞춤 해결 전략 (5가지)
   ↓
4️⃣ 💊 당신을 위한 긴급 처방전
```

## 📊 점수 계산 시스템

### 질문별 배점 (총 25점)

| 질문 | A | B | C | D |
|------|---|---|---|---|
| 1. 새벽 연락 | 0점 | 1점 | 4점 | 5점 |
| 2. 2시간 지각 | 0점 | 2점 | 4점 | 5점 |
| 3. "좋은 사람" 빈도 | 0점 | 2점 | 5점 | - |
| 4. 답장 늦음 | 0점 | 1점 | 2점 | 5점 |
| 5. 데이트 계획 | 0점 | 2점 | 4점 | 5점 |

### 레벨 분류

| 레벨 | 점수 범위 | 개수 | 특징 |
|------|----------|------|------|
| **LEVEL 1** | 0~10점 | 11개 (44%) | 고착화된 프렌드존 |
| **LEVEL 2** | 11~19점 | 9개 (36%) | 착한 남자 적신호 |
| **LEVEL 3** | 20~25점 | 6개 (24%) | 프레임 설계자 가능성 |

### LEVEL 3 추가 조건
- **0점 또는 1점 답변이 하나라도 있으면 자동 탈락**
- 20~25점이어도 최악 답변 있으면 → LEVEL 2

## 🎨 디자인 특징

### 색상 시스템
- **Primary**: `#FF3B3B` (빨강 - 긴급성)
- **Secondary**: `#FEE500` (카카오 옐로우)
- **Success**: `#4CAF50` (녹색 - 희망)
- **Warning**: `#FF9800` (주황 - 경고)

### 섹션 디자인
- **공감 섹션**: 흰색 배경 + 회색 좌측 보더 (진지함)
- **반론 차단 섹션**: 흰색 배경 + 회색 좌측 보더 (전문성)
- **희망 메시지**: 녹색 그라데이션 배경

### 반응형 디자인
- 모바일 퍼스트 접근
- `clamp()` 함수를 활용한 유동적 폰트 크기
- 768px 브레이크포인트

## 📈 전환율 최적화 전략

### 1. 심리적 방어기제 해제
- **공감 섹션**: 사용자의 자존감을 보호하며 수용
- **통합 반론 차단**: "5개 설문으로 판단?" 의문 선제 해소

### 2. 고통-희망-솔루션 흐름
- 현재 상태 진단 → 미래 고통 예측 → 희망 제시 → 응급처치 → 근본 해결책

### 3. 레벨별 맞춤 메시지
- **LEVEL 1**: 최악 답변 인용 + 패턴 지적
- **LEVEL 2**: 최악 답변 인용 + 추락 방지
- **LEVEL 3**: 최상위 답변 인용 + 칭찬 및 더 높은 단계 제시

### 4. 엄격한 LEVEL 3 기준
- 최악 답변 하나라도 있으면 차단
- 진짜 우수한 사람만 LEVEL 3 → 자존심/동기부여 증가
