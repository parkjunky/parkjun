## 📋 프로젝트 개요

이 프로젝트는 박준 코치의 전자책 『프레임 설계자』의 2차 판매를 위한 사전예약(얼리버드 알림) 랜딩 페이지입니다.

### 🎯 주요 목표

- 2차 판매 얼리버드 알림 신청 유도
- 1차 판매 완판 경험과 고객 후기를 통한 신뢰 구축
- 제품의 가치와 혜택을 명확하게 전달
- 고전환율 달성을 위한 최적화된 UX/UI 구현

## 📝 변경 이력

### Version 1.5.6 (2025-01-02) - 이미지 중앙 정렬 수정
- 🎨 **책 표지 이미지 중앙 정렬 개선**
  - **문제**: 첫 화면의 프레임 설계자 책 표지 이미지가 중앙 정렬 안 됨
  - **해결**:
    - 컨테이너에 `display: flex`, `justify-content: center` 적용
    - 이미지에 `display: block`, `margin: 0 auto` 추가
    - CSS에 `.main-hero img[alt*="전자책 표지"]` 선택자로 강제 중앙 정렬
    - 모바일에서 `max-width: 280px`로 최적화
