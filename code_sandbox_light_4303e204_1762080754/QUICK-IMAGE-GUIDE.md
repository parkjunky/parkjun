## 📐 사양

- **파일 크기**: 500KB 이하
- **포맷**: PNG (투명배경/텍스트), JPG (사진)
- **해상도**: 72-96 DPI
- **파일명**: 소문자, 하이픈(-) 사용, 영문+숫자만

---

## 🛠️ 준비 방법

### 후기 이미지 (review-*.png/jpg)

**카카오톡 스크린샷**
1. 실제 대화 캡처
2. 개인정보 가리기 (이름, 프로필 사진)
3. 핵심 문장 형광펜 표시 (선택)
4. 700x900px로 리사이즈

**이메일 후기**
1. 이메일 캡처
2. 발신자/수신자 익명 처리
3. 핵심 문장 강조
4. 800x600px로 리사이즈

**텍스트 이미지**
1. Canva, Figma 등에서 제작
2. 배경: 흰색/연한 색상
3. 폰트: Pretendard, Noto Sans KR
4. 여백 충분히 확보
