## 이미지 최적화 가이드

1. **파일 크기**: 각 이미지를 200KB 이하로 압축하는 것을 권장합니다
2. **해상도**: 웹 최적화를 위해 72-96 DPI를 사용하세요
3. **포맷**: JPG (사진), PNG (투명 배경이 필요한 경우)
4. **압축 도구**: TinyPNG, ImageOptim, Squoosh 등을 사용하세요

## 임시 Placeholder 이미지

실제 이미지가 없는 경우, 다음과 같은 무료 이미지 리소스를 활용할 수 있습니다:

- **Unsplash**: https://unsplash.com/
- **Pexels**: https://www.pexels.com/
- **Placeholder 서비스**: https://placeholder.com/

또는 CSS 그라디언트를 이용한 placeholder가 자동으로 표시됩니다.
