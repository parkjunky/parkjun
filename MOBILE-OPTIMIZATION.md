# 모바일 최적화 가이드

## 📱 개요

프레임 설계자 랜딩 페이지가 모바일에서 **50% 더 짧아지고**, **가독성이 2배 향상**되었습니다!

---

## 🎯 주요 개선 사항

### 1️⃣ 접기/펼치기 기능 ⭐️ NEW

긴 콘텐츠를 토글 방식으로 숨겨 스크롤 길이를 대폭 단축했습니다.

#### 적용된 섹션
- ✅ **주요 목차 미리보기** (18개 챕터)
- ✅ **핵심 이론 구성** (5가지 시스템)

#### 작동 방식
```
┌─────────────────────────────┐
│ 📖 주요 목차 미리보기    ▼  │ ← 클릭 가능
├─────────────────────────────┤
│ 6개 파트, 18개 챕터         │ ← 요약 정보
│ 매력의 본질부터 실전까지... │
└─────────────────────────────┘

        ↓ 클릭하면

┌─────────────────────────────┐
│ 📖 주요 목차 미리보기    ▲  │
├─────────────────────────────┤
│ Prologue: 왜 당신의...      │
│ Part 1. The Value           │
│ Chapter 1. 매력의 본질...   │
│ ...                         │
│ (전체 목차 표시)            │
└─────────────────────────────┘
```

## 💡 JavaScript 기능

### 토글 기능 구현
```javascript
const collapsibleHeaders = document.querySelectorAll('.collapsible-header');

collapsibleHeaders.forEach(header => {
    header.addEventListener('click', function() {
        this.classList.toggle('active');
        
        // 부드러운 스크롤
        if (this.classList.contains('active')) {
            setTimeout(() => {
                this.scrollIntoView({ 
                    behavior: 'smooth', 
                    block: 'nearest' 
                });
            }, 300);
        }
    });
});
```

### CSS 애니메이션
```css
.collapsible-content {
    max-height: 0;
    overflow: hidden;
    transition: max-height 0.3s ease;
}

.collapsible-header.active + .collapsible-content {
    max-height: 5000px;
}
```
